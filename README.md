# 🚀 Sentiment Analysis Using RNN

A deep learning project that performs sentiment analysis on text data using a Recurrent Neural Network (RNN). The model learns sequential patterns in sentences to classify them as positive or negative. It uses embedding layers and RNN architecture to capture contextual meaning in text.

---

## ✨ Overview

This project demonstrates how deep learning can understand human language by learning the order and meaning of words in a sentence.

The model uses:
- Embedding Layer to convert words into vectors
- Simple RNN to learn sequence patterns
- Dense layer to predict sentiment

---

## 🧠 Model Architecture

Input Text → Tokenization → Padding → Embedding Layer → SimpleRNN → Dense Layer → Output (Positive / Negative)

---

## ⚙️ Features

- Text preprocessing using tokenization
- Word embedding representation
- Sequential learning using RNN
- Binary sentiment classification
- Simple and beginner-friendly implementation

## 📌 Applications

---

## 🚀 How It Works

1. Text is converted into numerical sequences
2. Sequences are padded to equal length
3. Embedding layer learns word representations
4. RNN captures sequence relationships
5. Dense layer predicts sentiment output
